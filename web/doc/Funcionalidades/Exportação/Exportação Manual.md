## Visão Geral

A exportação manual permite ao utilizador gerar e descarregar relatórios sob demanda, com controlo total sobre o que exporta e em que formato.

---

## Formatos Disponíveis

| Formato | Descrição |
|---------|-----------|
| **PDF** | Relatório formatado para leitura/partilha |
| **Excel (.xlsx)** | Download local para análise em folha de cálculo |
| **Google Sheets** | Integração directa com uma folha Google |
| **Obsidian (.md)** | Markdown para importar na vault pessoal |
| **CSV** | Formato universal para ferramentas de dados |

---

## O que Pode Exportar

### Metas
- Dados de uma meta específica (com o seu relatório de encerramento)
- Dados de todas as metas de um período

### Serviços
- Relatório detalhado de um serviço específico

### Relatório Geral
- Dados gerais por semana, mês ou ano

---

## Selecção Granular de Períodos

O utilizador não está limitado a exportar períodos contíguos. Pode seleccionar períodos específicos de forma não contígua:

**Exemplo — Semanas:**
```
Semana 1  ✅ seleccionada
Semana 2  ❌ não incluída
Semana 3  ✅ seleccionada
Semana 4  ❌ não incluída
Semana 5  ✅ seleccionada
```
Resultado: relatório apenas com as semanas 1, 3 e 5.

**Aplica-se a:**
- Semanas específicas dentro de um mês/ano
- Meses específicos dentro de um ano
- Anos específicos
